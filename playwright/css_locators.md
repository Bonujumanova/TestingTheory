CSS локаторы используются для поиска элементов на веб-странице и являются основой для создания стилей и выполнения автоматизированных тестов

## Основы CSS-селекторов

CSS-селекторы позволяют выбрать HTML-элементы для их стилизации или взаимодействия в тестах. Селекторы можно разделить на несколько видов:

1. **Селекторы по тегу (Tag selectors)**

Выбирают все элементы указанного тега на странице. Например, если нужно выбрать все параграфы, используется селектор `p`:
```
p {
    color: blue;
}
```

Этот селектор применит синий цвет ко всем элементам `<p>`.

2. **Селекторы по классу (Class selectors)**
   
Классы назначаются для группировки элементов, и селекторы по классу позволяют выбрать элементы с конкретным классом. Класс задается с помощью точки:
```
.button {
         background-color: red;
}
```
Здесь все элементы с классом `button` будут с красным фоном.

4. **Комбинированные селекторы (Combinator selectors)**

Позволяют выбрать элементы в зависимости от их расположения относительно других элементов.
- **Дочерние элементы (Child combinator)** – выбирает элементы, которые являются непосредственными дочерними элементами другого:
    
```
div > p {
    color: green;
}
```

Это применит стиль ко всем `<p>`, которые находятся внутри любого `<div>`, на любом уровне вложенности.

5. **Атрибутные селекторы(Attribute selectors)**

Эти селекторы выбирают элементы по их атрибутам. Например, чтобы выбрать все `<input>` с атрибутом `type="text"`, используется следующий селектор:

```
input[type="text"] { 
    border: 1px solid black;
}
```
Этот селектор применит рамку ко всем текстовым полям ввода.

6. **Псевдоклассы (Pseudo-classes)**

Псевдоклассы выбирают элементы на основе их состояния, например, при наведении курсора:

```
a:hover { 
    color: orange; 
}
```
Здесь цвет всех ссылок изменится на оранжевый при наведении на них мыши.

7. **Псевдоэлементы (Pseudo-elements)**

 Псевдоэлементы позволяют стилизовать определенные части элемента. 
 
 Например:
```
p::first-line { 
    font-weight: bold; 
}
```
Этот селектор сделает жирным только первую строку всех параграфов.

## Составные селекторы

Составные CSS-селекторы позволяют выбрать элементы с учетом комбинации нескольких условий. Вот несколько примеров:

- **Комбинация класса и тега:**
```
button.primary { 
    background-color: green; 
}
```
Этот селектор применит зеленый фон к тегу `<button>` с классом `primary`.

- **Комбинация атрибутов и псевдоклассов:**
```
input[type="email"]:focus { 
    border-color: blue; 
}
```
Здесь применяется синий цвет границы ко всем полям ввода для email, когда на них установлен фокус.
